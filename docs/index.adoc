== Authentication
=== JWT-AUTH
|===
|*Header* |Authorization |JWT 혹은 User쪽 API를 통해 발급받은 Paper 자체 accessToken을 +
                            Bearer 형식으로 담아 보내십시오
|*Cookie* |Paper-Refresh-Token |JWT 혹은 User쪽 API를 통해 발급받은 Paper 자체 refreshToken을 +
                                쿠키에 담아 보내십시오 +
                                refreshToken은 발급될 때에 Set-Cookie로 발급되어 전송되므로, 따로 신경쓰지 않으셔도 됩니다
|===

=== JWT-AUTH-REFRESH
|===
|*Cookie* |Paper-Refresh-Token |JWT 혹은 User쪽 API를 통해 발급받은 Paper 자체 refreshToken을 +
                                쿠키에 담아 보내십시오 +
                                refreshToken은 발급될 때에 Set-Cookie로 발급되어 전송되므로, 따로 신경쓰지 않으셔도 됩니다
|===
